### settings.py
- TIME_ZONE을 'Asia/Seoul'로 변경
- USE_TZ을 False로 변경

```python
TIME_ZONE = 'Asia/Seoul'

USE_I18N = True

USE_L10N = True

USE_TZ = False
```
