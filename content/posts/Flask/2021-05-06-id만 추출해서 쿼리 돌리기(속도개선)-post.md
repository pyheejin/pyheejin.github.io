## query

- *로 전체 데이터를 가져오는 것 보다 id만 추출해서 가져오는게 훨씬 빠름
- 단, limit가 걸려있을 땐 데이터의 개수가 정해져 있으므로 별 차이 안날수도 있음

```python
# SQLAlchemy
query = Model.query.filter(Model.status == 1)
query = db.session.query(Model.id).filter(Model.status == 1)

# sql
select * from model;
select id from model;
```
